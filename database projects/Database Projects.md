# Database Projects: Complete Specifications

## Project 1: HR Management System

### Data That Must Be Stored
- Employee personal information
- Job title, department, hire date
- Salary and employment status
- Attendance records
- Leave requests

### How Data Will Be Used
- HR tracks employees and updates records
- Managers check attendance and leave status
- Payroll uses salary + attendance data

### Operations Needed
- Add/update employee
- Record attendance
- Submit/approve leave
- Generate reports (employee list, attendance summary)
